# LinkedIn Match - Privacy Policy & Terms of Service

**Last Updated:** January 20, 2026

---

## 1. Overview

LinkedIn Match ("the Extension") is a Chrome browser extension that provides AI-powered compatibility scoring for LinkedIn connection recommendations. This document outlines our privacy practices, data handling, and third-party service usage.

---

## 2. Technology Stack & Third-Party Services

### 2.1 Technologies Used

| Component | Technology | Purpose |
|-----------|------------|---------|
| Frontend | JavaScript, HTML, CSS | User interface and LinkedIn page interaction |
| AI/ML Scoring | Groq API (LLaMA 3.3 70B) | Generating compatibility scores and explanations |
| Backend Proxy | Node.js + Express | Secure API key management |
| Hosting | Railway.app | Proxy server hosting |
| Storage | Chrome Storage API | Local caching of user preferences |

### 2.2 Third-Party Services

#### **Groq API**
- **Provider:** Groq, Inc.
- **Model Used:** LLaMA 3.3 70B Versatile
- **Purpose:** AI-powered analysis of profile compatibility
- **Data Sent:** Profile headlines, skills, experience summaries (see Section 3)
- **Groq Privacy Policy:** https://groq.com/privacy-policy/

#### **Railway.app**
- **Purpose:** Hosting secure proxy server
- **Data Processed:** API requests are routed through proxy (no data stored)

---

## 3. Data Collection & Processing

### 3.1 Data We Collect

| Data Type | Source | Purpose | Storage |
|-----------|--------|---------|---------|
| Your LinkedIn Profile | User input / Auto-detected | Baseline for compatibility comparison | Local (Chrome Storage) |
| Target Profile Data | LinkedIn page (public info) | Compatibility scoring | Not stored (processed only) |
| Compatibility Scores | Groq API response | Display to user | Cached locally (24 hours) |
| User Feedback | User input (thumbs up/down) | Model improvement | Local storage only |

### 3.2 Data Sent to Groq API

When analyzing a LinkedIn profile, the following **publicly visible** information is sent to Groq:

```
- Your profile: name, headline, skills, experience years, industry
- Target profile: name, headline, skills, experience, education, location
```

**We DO NOT send:**
- ❌ LinkedIn login credentials
- ❌ Private messages
- ❌ Email addresses
- ❌ Phone numbers
- ❌ Connection lists
- ❌ Any non-public profile information

### 3.3 Data Retention

| Data | Retention Period |
|------|------------------|
| Cached scores | 24 hours (local) |
| User preferences | Until user clears extension data |
| API logs (proxy) | Not retained |

---

## 4. Data Security

### 4.1 Security Measures

- ✅ **No API Keys in Extension:** Groq API key stored on secure proxy server only
- ✅ **HTTPS Only:** All API communications encrypted
- ✅ **No Server-Side Storage:** Proxy server does not store any user data
- ✅ **Local Processing:** Most data processing happens in your browser

### 4.2 Proxy Server Security

The proxy server (hosted on Railway.app):
- Acts as a pass-through for API requests
- Does not log or store request/response data
- API key is stored as environment variable (not in code)
